# Páginas legales — Sueca Online

Contiene la Política de Privacidad y los Términos y Condiciones para Sueca Online.

## Publicación en GitHub Pages

1. Crea un repositorio `sueca-game-legal` en GitHub (o usa el existente).
2. Sube estos archivos: privacy.html, terms.html, support.html, index.html, logo.png.
3. Activa GitHub Pages en Settings → Pages → source: main branch.
4. La URL será: `https://<usuario>.github.io/sueca-game-legal/`

Para Sueca Game se usa: `https://danipar7777.github.io/`

## Archivos

- **privacy.html** — Política de Privacidad
- **terms.html** — Términos y Condiciones
- **support.html** — Soporte (URL para App Store Connect)
- **index.html** — Índice con enlaces
- **logo.png** — Logo (copiar desde `../assets/logo.png`)

## Configuración en la app

En `.env`:
```
EXPO_PUBLIC_LEGAL_BASE_URL=https://danipar7777.github.io
```
