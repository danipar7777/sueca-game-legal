2. Datos que recogemos

Al usar Sueca Online podemos recoger:

  • Cuenta: identificador de Apple Sign-In, correo electrónico (si lo proporcionas), nombre para mostrar que elijas en el juego.
  • Datos de uso: partidas jugadas, créditos, puntos, nivel y estadísticas para rankings.
  • Publicidad (AdMob): usamos Google AdMob para mostrar anuncios. Antes de mostrar anuncios personalizados te pedimos tu consentimiento (App Tracking Transparency en iOS). Puedes aceptar o rechazar; si rechazas, seguirás viendo anuncios pero menos adaptados a tus intereses. AdMob puede recoger el identificador publicitario (IDFA), datos de uso e interacción con anuncios para medición y personalización.
  • Red local: Google AdMob puede acceder a la red local para la entrega de anuncios. Este acceso no recoge datos personales adicionales.

3. Finalidad

Usamos tus datos para: gestionar tu cuenta, permitirte jugar partidas multijugador, mostrar rankings y recompensas, mostrar publicidad y mejorar la app.

4. Bases legales

El tratamiento se basa en la ejecución del servicio que solicitas, el consentimiento (cuando aplique) y el interés legítimo para mejorar la app.

5. Conservación

Conservamos los datos mientras mantengas la cuenta activa. Puedes solicitar la eliminación contactando a mixifystudioapps@gmail.com.

6. Derechos

Tienes derecho a acceder, rectificar, suprimir, oponerte y limitar el tratamiento. Para ejercerlos: mixifystudioapps@gmail.com. Ante la AEPD puedes presentar reclamación.

7. Seguridad

Implementamos medidas técnicas y organizativas para proteger tus datos.

8. Cambios

Podemos actualizar esta política. Los cambios relevantes se publicarán aquí con la nueva fecha de actualización.
